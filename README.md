# 🌍 Procedural Planet Generator

**Комп'ютерний проєкт з Дискретної математики (ІІ семестр, APPS UCU)**
**Тема:** Теорія автоматів і симуляція штучного життя (Експериментальний проєкт: Процедурна генерація ландшафту).

Цей проєкт — це генератор тривимірних планет земного типу. Замість звичайної 2D-карти, програма генерує безшовний сферичний ландшафт на основі заданого початкового значення (`seed`). Проєкт поєднує строгі математичні алгоритми, оптимізовані обчислення на Python та сучасний веб-інтерфейс для 3D-візуалізації.

---

## ✨ Основні можливості

* **100% Процедурна генерація:** Кожен `seed` створює унікальну, але детерміновану планету з континентами, океанами та кліматичними зонами.
* **Сферичний 3D-шум:** Відсутність спотворень на полюсах завдяки використанню просторового шуму замість натягування 2D-текстури на сферу.
* **Два режими роботи:**
    * **CLI (Command Line Interface):** Повноцінне керування через термінал за допомогою `argparse` для швидкої генерації масивів даних.
    * **Web GUI:** Інтерактивний 3D-переглядач у браузері з естетичним сучасним інтерфейсом (OLED-темна тема, неонові акценти), написаний на React + Three.js.
* **Реальний час:** Швидка перегенерація ландшафту "на льоту" завдяки JIT-компіляції (Numba) та оптимізованій роботі з пам'яттю.

## 🧮 Математична та алгоритмічна база

Згідно з вимогами проєкту, ми використали наступні концепції дискретної математики та алгоритмічного моделювання:

### 1. 3D-шум (Просторова генерація)
Для створення базового рельєфу (карти висот) використовується фрактальний шум (наприклад, Simplex або Perlin noise). Замість обчислення функції у просторі $f(x, y)$, ми обчислюємо її на поверхні тривимірної сфери $f(x, y, z)$, де координати нормалізовані до радіуса планети. Спектральний синтез (накладання кількох "октав" шуму з різними частотами та амплітудами) дозволяє створити як масивні континенти, так і дрібні деталі гірських хребтів.

### 2. Клітинні автомати (Тектоніка та біоми)
Поверхня планети розбивається на дискретну сітку (наприклад, використовуючи сферичні комірки). Клітинні автомати застосовуються для:
* **Симуляції ерозії та тектоніки:** Сусідні клітинки взаємодіють одна з одною за певними правилами (наприклад, перенесення маси з вищих точок у нижчі).
* **Формування клімату:** Температура спадає від екватора до полюсів, а вологість поширюється від океанів. Стан кожної клітинки (її біом: пустеля, ліс, тундра) визначається як функція від станів її сусідів на попередніх ітераціях.

## 🛠 Технологічний стек

* **Core (Backend):** Python 3, `argparse` (CLI), Numba (оптимізація обчислень), FastAPI (REST API для передачі даних на фронтенд).
* **Frontend (GUI):** idk, Tailwind CSS.
* **Архітектура:** Розділення логіки обчислень та візуалізації. Python-скрипт генерує карти висот та біомів (текстури), які передаються на клієнт для фізичного витискання 3D-сфери (`displacement map`).

## 🚀 Запуск проєкту

Проєкт розроблено з урахуванням кросплатформності (повна підтримка Linux, macOS, Windows).

### Встановлення залежностей
```bash
git clone <repository-url>
cd planet_generator
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Запуск CLI
```bash
python src/main.py --seed 42 --resolution 1024 --output planet.png
```

### Запуск Web GUI (Інтерактивний режим)
```bash
docker compose up --build
```
